DESCRIPTION

r.out.vic_met creates a set of VIC hydrological model meteorological input ascii files. Files are $prefix_latitude_longitude.dat. Format is: "%.2f %.2f %.2f\n" with: double precipitation double Tmax double Tmin This is an input to VIC (Variable Infiltration Capacity) in forcing/ of TEST example.
