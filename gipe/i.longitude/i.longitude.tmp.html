NAME

i.longitude - creates a longitude map

KEYWORDS

longitude, projection

SYNOPSIS

i.longitude
i.longitude help
i.longitude [-q] input=string longitude=string [--overwrite] [--verbose] [--quiet]

Flags:

-q
Quiet
--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

input=string
Name of the input map
Default: input
longitude=string
Name of the output longitude layer
Default: longitude

DESCRIPTION

i.latitude creates a latitude (degree decimal) map from any map in any projection using PROJ.4 library. This is an input to r.sun and i.evapo.potrad
