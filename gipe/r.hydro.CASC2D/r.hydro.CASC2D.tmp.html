NAME

r.hydro.CASC2D - Fully integrated distributed cascaded 2D hydrologic modeling.

KEYWORDS

SYNOPSIS

r.hydro.CASC2D
r.hydro.CASC2D help
r.hydro.CASC2D [-smtoepidbuq] [watershed_mask=string] elevation=string [initial_depth=string] [storage_capacity=string] [interception_coefficient=string] [roughness_map=string] [conductivity=string] [capillary=string] [porosity=string] [moisture=string] [pore_index=string] [residual_sat=string] [lake_map=string] [lake_elev=string] [radar_intensity_map=string] [links_map=string] [nodes_map=string] [channel_input=string] [table_input=string] [dis_profile=string] [wat_surf_profile=string] [hyd_location=string] [r_gage_file=string] outlet_eastNnorthNslope=east,north,bedslope [Manning_n=float] [unif_rain_int=float] [num_of_raingages=integer] time_step=integer [gage_time_step=integer] [radar_time_step=integer] rain_duration=integer tot_time=integer [write_time_step=integer] [unit_el_conv=float] [unit_lake=float] [unit_space=float] [d_thresh=float] discharge=string [dis_hyd_location=string] [depth_map=string] [inf_depth_map=string] [surf_moist_map=string] [rate_of_infil_map=string] [dis_rain_map=string] [--verbose] [--quiet]

Flags:

-s
do not check square tolerance
-m
r_gage_file units in mm/hr
-t
interpolates raingage rainfall intensities using Thiessen polygon technique (default: inverse square distance)
-o
routes edge-accmulated overland flow out of active region (ONLY when no mask is specified)
-e
performs explicit channel routing
-p
assumes uniform channel geometry in each link (needs -e option)
-i
performs Priessman double sweep implicit channel routing
-d
performs only drainage of the basin for implicit routing by flooding the basin
-b
determines initial flow depths for implicit routing by performinf standard step backwater computations
-u
print discharges in cfs and volumes in cubic ft
-q
skips printing iteration, time, and discharge values to the screen
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

watershed_mask=string
map of watershed boundary (or mask); recommended
elevation=string
map of elevation
initial_depth=string
map of initial overland (not lakes) depth in mm
storage_capacity=string
map of vegetation storage capacity in tenths of mm
interception_coefficient=string
map of interception coefficient (values in 1000*actual coefficient)
roughness_map=string
map of surface roughness coefficient (values in 1000*Manning n)
conductivity=string
map of soil saturated hydraulic conductivity in tenths of mm/hr
capillary=string
map of soil capillary pressure head at the wetting front in tenths of mm
porosity=string
map of soil effective porosity (values in 1000*porosirty)
moisture=string
map of soil initial moisture (values in 1000*moisture)
pore_index=string
map of soil pore-size distribution index (Brooks-Corey lambda) in 1000*index
residual_sat=string
map of soil residual saturation (values in 1000*residual saturation)
lake_map=string
map of lakes categories
lake_elev=string
map of lakes initial water surface elevation (for unit see unit_lake)
radar_intensity_map=string
map of radar- (or otherwise-) generated time series of rainfall intensity in mm/hr
links_map=string
map of channel network link numbers
nodes_map=string
map of channel network node numbers
channel_input=string
channel input data file name (ASCII)
table_input=string
look-up table file for links with breakpoint cross section (ASCII)
dis_profile=string
channel initial discharge profile file name (ASCII)
wat_surf_profile=string
channel intial water surface profile file name (ASCII)
hyd_location=string
file name containing link and node addresses of internal locations where discharge hydrographs are to be saved (ASCII)
r_gage_file=string
raingage rainfall input file name (ASCII), intensities in in/hr
outlet_eastNnorthNslope=east,north,bedslope
easting, northing, and bed slope at the outlet cell
Manning_n=float
spatially uniform overland Manning n roughness value
unif_rain_int=float
spatially uniform rainfall intensity in mm/hr
num_of_raingages=integer
number of recording raingages
time_step=integer
computational time step duration in sec
gage_time_step=integer
time step of recorded raingage data in sec
radar_time_step=integer
time increment between radar- (or otherwise) generated rainfall maps in sec
rain_duration=integer
total rainfall duration in sec
tot_time=integer
total simulation time or channel drainage time (-d option) in sec
write_time_step=integer
time increment for writing output raster maps in sec
unit_el_conv=float
unit convertion factor by which the values in elevation map must be DIVIDED to convert them into meters
unit_lake=float
unit convertion factor by which the values in lake surface elevation map must be DIVIDED to convert them into meters
unit_space=float
unit convertion factor by which the region easting and northing values must be DIVIDED to convert them into meters
d_thresh=float
threshold overland depth, in meters, below which overland routing will not be performed (i.e. average depression storage)
discharge=string
outflow discharge output file name (ASCII)
dis_hyd_location=string
output file name for discharge hydrograph at internal locations (ASCII)
depth_map=string
output map of surface depth in mm
inf_depth_map=string
output map of cumulative infiltration depth in tenth of mm
surf_moist_map=string
output map of surface soil moisture in number of fractions of a thousand
rate_of_infil_map=string
output map of infiltration rate in mm/hr
dis_rain_map=string
output map of distributed rainfall intensity in mm/hr
