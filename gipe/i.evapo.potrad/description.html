DESCRIPTION

i.evapo.potrad Calculates the diurnal potential evapotranspiration after bastiaanssen (1995). This is converting all Net radiation from the diurnal period into ET. It takes input maps of Albedo, surface skin temperature, latitude, day of year, single-way transmissivity and takes input value of the density of fresh water. The "-r" flag permits output map of Diurnal net radiation to use in r.eb.eta.
