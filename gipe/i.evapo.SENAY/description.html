DESCRIPTION

i.evapo.SENAY Calculates the diurnal actual evapotranspiration after Senay (2007). This is converting all Net radiation from the diurnal period into ET, then uses Senay equation for evaporative fraction. It takes input maps of Albedo, surface skin temperature, latitude, day of year, single-way transmissivity and takes input value of the density of fresh water. DEM is used for calculating min and max temperature for Senay equation. The "-s" flag permits output map of evaporative fraction from Senay.
