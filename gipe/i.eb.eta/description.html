DESCRIPTION

i.eb.eta calculates the actual evaporation (mm) for diurnal period after bastiaanssen (1995). It takes input of Diurnal Net Radiation (see r.sun), evaporative fraction (see r.eb.evapfr) and surface skin temprature.
