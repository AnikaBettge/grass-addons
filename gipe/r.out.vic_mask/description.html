DESCRIPTION

r.out.vic_mask creates a VIC hydrological model watershed basin mask input ascii file. This is used by gstation.c and select_stations.c to grid meteorological datasets into VIC. http://www.hydro.washington.edu/Lettenmaier/Data/PRCP.html#A4.5 This is an input to VIC (Variable Infiltration Capacity).
