DESCRIPTION

Imports a stream of ASCII x,y[,z] coordinates as a line or series of lines.

NOTES

Input ASCII coordinates are simply a series of "x y" data points. Lines are separated by a row containing "NaN NaN". Line categories start at 1 and increase sequentially.

You can import 3D lines by providing 3 columns of data in the input stream and using the -z flag.

This script is a simple wrapper around the v.in.mapgen module.
