DESCRIPTION

v.in.marxan is a python script reads output from marxan and updates the planning grid vector file prepared using v.out.marxan.

NOTES

Use of the DOS output format will create a second file with '.dos' appended to the output file name. Consistent specification of the Planning Unit Key field is necessary in all Marxan scripts to ensure a working set of ouptut files. All GRASS Marxan modules require Python, PostgreSQL 8.x or higher and are intended for use with Marxan version 1.8.10.
