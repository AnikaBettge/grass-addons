DESCRIPTION

r.out.mbtiles_prep creates a TMS tileset and support files ready to load into software like Geopaparazzi which will make the final conversion to a mbtiles spatialite database file.

NOTES

Either an individual raster map or a raster map imagery group may be given as input.

If not specified the zoom level will be automatically chosen.

EXAMPLE

Creates a tileset for the North Carolina sample dataset's soilsID raster map at automatically chosen zoom levels 11-14. 
g.region rast=soilsID
r.out.mbtiles_prep in=soilsID out=soil_tiles format=jpeg
