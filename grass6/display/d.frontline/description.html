NAME

d.frontline - Display different types of frontlines on the graphic monitor using d.graph (optionally save graph file and ps.map file)

KEYWORDS

display, graphics, vector, symbology

SYNOPSIS

d.frontline
d.frontline help
d.frontline [-ms] map=name [where=string] [hcolor=string] [list=string] [layer=integer] [side=string] interval=float symbol=string [size=integer] [line_color=string] [fill_color=string] [width=integer] [graph=name] [psmap=name] [--verbose] [--quiet]

Flags:

-m
Select vector features interactively (with a mouse)
-s
Save graphics commands in file(s) (for use with d.graph and/or ps.map)
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

map=name
Name of input vector map
where=string
WHERE conditions of SQL query statement without 'where' keyword
The WHERE statement for data filtering
hcolor=string
Highlight color for vector interactive selecting
Standard color name or R:G:B
Default: yellow
list=string
Category number(s)
Example: 1,3,7-9,13
layer=integer
Layer number. If -1, all layers are extracted.
A single vector map can be connected to multiple database tables. This number determines which table to use.
Default: 1
side=string
Side of the frontline
Options: left,right,both
Default: left
interval=float
Interval between symbols (in mapset units)
symbol=string
Type of symbol (<front/...> symbols will be produced by script at first use: front/arrow_int, front/arrow_ext, front/box, front/circle, front/half_box, front/line, front/triangle). May be any symbol from $GISBASE/etc/symbol/ or $MAPSET/symbol/ directories.
size=integer
Symbol size
Default: 20
line_color=string
Color for line (standard color name or R:G:B)
Default: black
fill_color=string
Color for fill (standard color name or R:G:B)
Default: grey
width=integer
Width of symbol's line
Default: 0
graph=name
Output file to save the d.graph commands
psmap=name
Output file to save the ps.map commands
