NAME

d.zoom.keys - Allows to change the current geographic region settings interactively, with a keyboard. Use arrow keys to navigate, "-" to make futher, "+" (without "Shift") to make closer.

KEYWORDS

display, zoom

SYNOPSIS

d.zoom.keys
d.zoom.keys help
d.zoom.keys [-txr] [mon=name] [magn=number] [--verbose] [--quiet]

Flags:

-t
Use navigation from terminal
-x
Use navigation in X-monitor (requires and )
-r
Decrease region resolution during navigation
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

mon=name
Name of graphics monitor to select (default is currently selected monitor).
magn=number
Magnification: default is "1" (1/10 of current region). "magn" > 1 gives positive magnification, "magn" < 1 gives negative one.
