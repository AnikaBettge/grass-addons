NAME

v.trimesh - Create a triangular mesh from a vector map

KEYWORDS

vector

SYNOPSIS

v.trimesh
v.trimesh help
v.trimesh [-a] input=name column=string output=name [--overwrite] [--verbose] [--quiet]

Flags:

-a
Do not use areal constraint
--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

input=name
Name of input vector map
column=string
Column name with area constraints/centroid attributes
output=name
Name for output vector map
