DESCRIPTION

i.ortho.corr create a new image, it search the images near the input map and the output contains part of near images where the camera angle value it's the best.

NOTES

You must create a tile index contains all the images to correct, you can use gdaltindex to create it 
gdaltindex tile.shp $GRASSDATA/$LOCATION/$MAPSET/cellhd/*imagery.ortho
The field option it's used for the field's name which contain the path to the file, the default is location that it is used by gdaltindex The exclude option it serve to remove some tiles, for example you have tile of different flight, you can pass a string or a regex

EXAMPLES

Create tile index 
gdaltindex tile.shp $GRASSDATA/$LOCATION/$MAPSET/cellhd/*imagery.ortho
Import tile index inside the mapset 
v.in.ogr dns=tile.shp out=tile_images
Start i.ortho.corr with the default parameters, the output map's name it will be image.ortho_corr. You can use default parameters if you didn't change the output prefix in i.ortho.photo 
i.ortho.corr input=image.ortho tiles=tile_images
Start i.ortho.corr with different parameters 
i.ortho.corr input=image.photo tiles=tile_images osuffix=.photo csuffix=.camera
