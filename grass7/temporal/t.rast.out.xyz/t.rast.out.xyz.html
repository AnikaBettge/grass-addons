DESCRIPTION

The t.rast.out.xyz module exports a space time raster dataset as a list of x,y,z values into an ASCII text file.

NOTES

This module will by default not export x,y coordinates for raster cells containing a NULL value. This includes cells masked by a raster MASK. Using the flag -i also these raster cells will be included in the exported data. t.rast.out.xyz is simply a front-end to "r.out.xyz".

EXAMPLE

t.rast.out.xyz input=mystrds output=/tmp/mystrds.csv
