DESCRIPTION

The input of this module is a single space time raster dataset, the output is a single raster map layer. A subset of the input space time raster dataset can be selected using the where option. The sorting of the raster map layer can be set using the sort option. Be aware that the sorting of the maps significantly influences the result of the patch. By default the maps are sorted by desc by the start_time so that the newest raster map in r.patch.

t.rast.patch is a simple wrapper for the raster module r.patch.

EXAMPLE

Patching the average temperature (filling missing pixels by subsequent maps in the time series): 
t.rast.patch input=tempmean_monthly output=tempmean_patched
