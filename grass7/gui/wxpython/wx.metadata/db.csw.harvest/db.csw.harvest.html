DESCRIPTION

Module db.csw.harvest allows to harvest metadata between two catalogs.

NOTES

For dependencies and installation instructions see wiki page. For importing metadata to local server transactions must be allowed. The parameter "transactions" is stored in configure file "default.cfq"(by default in pycsw installation foleder ).

EXAMPLES

Harvesting of remote cataloge to local: 

db.csw.harvest source=http://www.rndt.gov.it/RNDT/CSW destionantion=http://localhost:8000/
