NAME

r.soillossbare.py - Calculate annual soil loss [t/(ha*a)] for bare soil.

SCHLÜSSELWÖRTER

erosion,raster,bare soil, potential soilloss

SYNOPSIS

r.soillossbare.py
r.soillossbare.py help
r.soillossbare.py [-r] soillossbare=name flowaccmethod=name elevation=name resolution=float kfactor=name rfactor=name [fieldblock=name] [map=name] [flowacc=name] [constant_m=float] [constant_n=float] [--overwrite] [--verbose] [--quiet]

Flags:

-r
Remove intermediate results
--overwrite
Ausgabedateien dürfen bereits existierende Dateien überschreiben.
--verbose
Ausführlicher Ausgabemodus
--quiet
Schweigsamer Ausgabemodus

Parameter:

soillossbare=name
Output map (soilloss for ungrown soil)
flowaccmethod=name
Module for flowaccumulation calculation
Optionen: r.terraflow,r.flow,r.watershed
Standardwert: r.terraflow
elevation=name
Digital Elevation Model
resolution=float
Resolution of Digital Elevation Model (x y)
Standardwert: 2
kfactor=name
K-Factor (soil erodibility factor)
rfactor=name
R-Factor (rain erosivity factor)
fieldblock=name
Fieldblock raster map with NULL/0-values as barrier
map=name
Fieldblock vector map
flowacc=name
Flowaccumulation raster map (instead of calculation)
constant_m=float
RUSLE3D exponential m (0.2..0.6)
Standardwert: 0.4
constant_n=float
RUSLE3D exponential n (1.2..1.6)
Standardwert: 1.3
