DESCRIPTION

r.vect.stats bins points from a vector map into a raster map.

Use g.region to set the extent and resolution of the resulting raster.

EXAMPLES

Calculate for current region the beam irradiance (direct radiation) for DOY 355 in 2014 from 8am to 3pm: 
g.region vect=elev_lid792_bepts res=10

SEE ALSO

g.region, r.in.xyz, r.in.lidar
