DESCRIPTION

r.modis.import import into GRASS GIS Level 3 MODIS product.

NOTE

The input file is the list of full path to HDF files, one for line. The input file have to be inside the folder where the HDF files are.
The mrtpath option is the path to the main folder of Modis Reprojection Tools; it contain bin and data folder, this two folders are essential for a successful result.
It is not possible to use flag "q" and option "spectral" together.

Warning:
  • Currently it's tested and It work correctly only with Lat/Long
  • Currently the mosaic flag it is not working

EXAMPLES

Import file with the default subset without QA layer 
    r.modis.import filename=/path/to/listfile mrtpath=/path/to/mrt
Import file with the default subset with QA layer 
    r.modis.import -q filename=/path/to/listfile mrtpath=/path/to/mrt
Import file with the your own subset 
    r.modis.import filename=/path/to/listfile mrtpath=/path/to/mrt spectral="( 1 )"
