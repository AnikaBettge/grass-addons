DESCRIPTION

The r.modis suite is a toolset to import MODIS data in GRASS GIS. It use the pyModis library and the MODIS Reprojection Tool softwares to convert, mosaik and process MODIS data. Please check to have MODIS Reprojection Tool on your Operation System, pyModis it is distributed with r.modis program. The suite offers three modules for interface with MODIS data. Each modules it is useful for a specific operation, read below for more info. It can download several products, tiles and days of MODIS data from NASA FTP servers. After, it can import the 3 Level products as single image or daily mosaik or process the raw data.

NOTE

The first time that you use r.modis suite you have to do some easy operations to work corectly with r.modis modules.
  • Create a r.modis folder inside $HOME/.grass7
  • Create SETTING.py file inside $HOME/.grass7/r.modis and compile like below: 
        #!/usr/bin/python

    USERNAME='your_NASA_FTP_username'
    PASSWORD='your_NASA_FTP_password'
    MRTPATH='your_path_to_MRT_software'
$HOME/.grass7/r.modis is also the default path where r.modis suite creates some subfolder to store the different MODIS files. Each module permits to change the previous folder with another directory using an module's option.
