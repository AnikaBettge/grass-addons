DESCRIPTION

r.modis.download download MODIS products. It can download several tiles and days of each MODIS product.

NOTE

To set the username and password options you have to use setting option; you can use a file, in this case the first line is the username, usually is 'anonymous', in the second line the password, your e-mail address 
  your_NASA_FTP_username
  your_NASA_FTP_password
otherwise you can pass the values from standard input.
Warning: NASA policy don't permit to open more then ten connections with the FTP servers. Please don't run more 10 the r.modis.download process at the same time.
By default the files downladed are store in the path where is the setting file; you can change the directory with folder options. You have to set folder option if you send user and password by standard input
The time of download change with the number of tiles, days and the type of product, for example NDVI product it is from 5 MB to 270 MB and LST is from 2 MB to 21 MB.

EXAMPLES

To download LST Terra product using the default options and set the user and password by standard input you can launch 
    r.modis.download setting=-
Reading the user and password options from file 
    r.modis.download setting=/home/user/.grass/r.modis/SETTING
To download LST Terra product using the default options and change the starting and ending dates 
    r.modis.download setting=/home/user/.grass/r.modis/SETTING startday=2011-05-31 endday=2011-05-01
To downlad another product (for example Snow eight days), with other default options 
    r.modis.download setting=/home/user/.grass/r.modis/SETTING product=snow_terra_eight
To use r.modis.download in a script with one of the other module you have to set -g flag to return the name of the file containing a list of HDF names downladed 
    r.modis.download -g setting=/home/user/.grass/r.modis/SETTING startday=2011-05-31 endday=2011-05-01
