NAME

r.green.hydro.theoretical - Calculate the hydropower energy potential for each basin starting from discharge and elevation data. If existing plants are available it computes the potential installed power in the available part of the rivers.

KEYWORDS

raster

SYNOPSIS

r.green.hydro.theoretical
r.green.hydro.theoretical --help
r.green.hydro.theoretical [-d] elevation=name discharge=name [rivers=name] [lakes=name] [input=name] threshold=string [basins=name] [stream=name] output=name [segments=name] [hydro=name] [hydro_layer=string] [hydro_kind_intake=string] [hydro_kind_turbine=string] [other=name] [other_layer=name] [other_kind_intake=string] [other_kind_turbine=string] [output_segm=name] [output_point=name] [--help] [--verbose] [--quiet] [--ui]

Flags:

-d
Debug with intermediate maps
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

Required:

elevation=name [required]
Name of input elevation raster map
discharge=name [required]
Name of river discharge [l/s]

Optional:

rivers=name
Name of river network
lakes=name
Name of lakes

Discharge:

input=name
Name of vector points map with discharge data
Or data source for direct OGR access

Basin Potential:

threshold=string [required]
Minimum size of exterior watershed basin
Default: 0
OR if basins and streams were generated by r.watershed
basins=name
Name of basin map obtained by r.watershed
stream=name
Name of stream map obtained by r.watershed
output=name [required]
Name of vector map with basin potential

Residual Theoretical Potential:

segments=name
Name of vector map with river segment without plants
hydro=name
Name of input vector map
Name of the vector map with points of intakes and turbines.
hydro_layer=string
Layer number or name
Name of the vector map layer of the hydro plant, with the following attributes: kind (water intake/turbine), discharge [l/s], working hours [hours], altitute[m], id point, id plant
Default: 1
hydro_kind_intake=string
Value contained in the column: hydro_kind that indicate the plant is an intake.
Default: intake
hydro_kind_turbine=string
Value contained in the column: hydro_kind that indicate the plant is a turbine.
Default: turbine
other=name
Name of vector map
Name of the vector map point of other plants such as irrigation, acqueducts, etc.
other_layer=name
Name of input vector map
Name of the vector map layer of other plants, with the following attributes: kind (water intake/turbine), discharge [m3/year], id point, id plant
Default: 1
other_kind_intake=string
Value contained in the column: other_kind that indicate the plant is an intake.
Default: intake
other_kind_turbine=string
Value contained in the column: other_kind that indicate the plant is a turbine.
Default: turbine
output_segm=name
Name of output vector with potential segments
output_point=name
Name of output vector with potential intakes and restitution
