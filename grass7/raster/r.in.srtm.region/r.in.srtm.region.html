DESCRIPTION

r.in.srtm.region imports all SRTM tiles covering the current region or region extents given with region into GRASS, patches the tiles together and optionally interpolates holes for SRTM V2.1.

r.in.srtm.region downloads SRTM V3 tiles at 3 arc seconds resolution from:
http://e4ftl01.cr.usgs.gov/SRTM/SRTMGL3.003/2000.02.11/
SRTM V3 tiles at 1 arc second resolution from:
http://e4ftl01.cr.usgs.gov/SRTM/SRTMGL1.003/2000.02.11/
and SRTM V2.1 tiles from: http://dds.cr.usgs.gov/srtm/
or optionally uses a local folder with previously downloaded files if the local option is given.

NOTES

This module only operates in LatLong locations.

EXAMPLE

Import of SRTM V3 (1 arc seconds ~ 30m) covering the current region: 
# run in LatLong location
r.in.srtm.region -1 output=srtmV3 memory=2000
