DESCRIPTION

r.in.srtm.region imports all SRTM tiles covering the current region or region extents given with region into GRASS, patches the tiles together and optionally interpolates holes for SRTM V2.1. The SRTM V3 ("SRTM Plus") are already void-filled.

r.in.srtm.region downloads

  • SRTM V3 tiles at 1 arc second resolution from:
    http://e4ftl01.cr.usgs.gov/SRTM/SRTMGL1.003/2000.02.11/
  • SRTM V3 tiles at 3 arc seconds resolution from:
    http://e4ftl01.cr.usgs.gov/SRTM/SRTMGL3.003/2000.02.11/
  • SRTM V2.1 tiles at 3 arc second resolution from: http://dds.cr.usgs.gov/srtm/
  • or optionally it uses a local folder with previously downloaded files if the local option is given.
Importantly, for the SRTM tiles download a user registration is needed at https://urs.earthdata.nasa.gov/users/new

In the user profile, two specific applications must be approved in the "My application" tab:

  • LP DAAC Data Pool, and
  • Earthdata Search

NOTES

This module only operates in LatLong locations.

EXAMPLE

Import of SRTM V3 (1 arc seconds ~ 30m) covering the current computational region: 
# run in LatLong location
r.in.srtm.region -1 output=srtmV3 memory=2000
