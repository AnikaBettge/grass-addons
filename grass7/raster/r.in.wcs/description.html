NAME

r.in.wcs.py - Downloads and imports coverage from WCS server.

SCHLÜSSELWÖRTER

OGC web services

SYNOPSIS

r.in.wcs.py
r.in.wcs.py help
r.in.wcs.py [-c] url=string [coverage=string] [urlparams=string] [username=string] [password=string] [output=string] [location=string] [region=string] [--overwrite] [--verbose] [--quiet]

Flags:

-c
Get the server capabilities then exit
--overwrite
Ausgabedateien dürfen bereits existierende Dateien überschreiben.
--verbose
Ausführlicher Ausgabemodus
--quiet
Schweigsamer Ausgabemodus

Parameter:

url=string
Service URL (typically http://.../mapserv? )
Standardwert: http://...?
coverage=string
Coverage name to request
urlparams=string
Additional query parameters to pass to the server
username=string
Username for server connection
password=string
Password for server connection
output=string
Name for output raster map (default: coveragename)
location=string
Name der neu zu erstellenden Location.
region=string
Name for region instead of current region
