DESCRIPTION

v.in.gbif imports GBIF species distribution data. The location must be in latlon (WGS84).

The species distribution data downloaded from GBIF hast to be unzipped before importing the csv file.

v.in.gbif saves the data to a GDAL VRT - Virtual Datasource which will be imported by v.in.ogr. The VRT data set can be used in any GDAL aware software. As some column names in the original data set are similar to SQL reserverd key words, the columns will renamed with the prefix g_.

EXAMPLE

  v.in.gbif vector= cat=40102 dir=C:\tmp distance=25
  v.in.gbif input=C:\data\0004248-150811131857512\0004248-150811131857512.csv output_vrt=chondrilla.vrt dir=C:\data\0004248-150811131857512\ output=chondrilla
