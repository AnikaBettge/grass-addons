DESCRIPTION

v.fixed.segmenpoints creates segment points along a vector line with fixed distances by using the v.segment module. A category of 1 line has to be given. Start and end point of the line will be considered.

EXAMPLE

  # nc sampe data set
  v.fixed.segmentpoints vector=streams@PERMANENT cat=40102 dir=C:\tmp distance=25

DEPENDENCIES

  • v.segment
