DESCRIPTION

v.mkhexgrid creates a hexagonal grid vector map for the selected region.

NOTES

This module functions by internally creating a number of GRASS ASCII files and then importing and cleaning them.

EXAMPLE

North Carolina sample dataset: 
g.list region
v.mkhexgrid map=hexagon regionname=wake_30m hexarea=20000000 \
            method=area

Hexagon grid map 

v.mkhexgrid map=hexagon regionname=wake_30m sidelength=5000 \
            method=sidelength
