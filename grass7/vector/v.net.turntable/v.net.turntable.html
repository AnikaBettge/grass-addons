DESCRIPTION

v.net.turntable creates a turntable with the costs for every possible turn on every possible node (intersection, crossroad) in given layer (alayer). U-turns are taken in account too.

These modules support the turntable extension: v.net.alloc, v.net.iso, v.net.path, v.net.salesman, v.net.steiner.

For more information about turns in the vector network analyses see wiki page.

EXAMPLE

 v.net.turntable input=roadsmajor output=roadsmajor_ttb
