NAME

r.stream.pos - Route azimuth, direction and relation to streams of higher order

KEYWORDS

stream, order, route azimuth, route direction

SYNOPSIS

r.stream.pos
r.stream.pos help
r.stream.pos [-s] stream=string dir=string multipier=integer [cells=string] [lengths=string] [--overwrite] [--verbose] [--quiet]

Flags:

-s
Create new stream category sequence
--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

stream=string
Name of stream mask input map (r.watershed or r.stream.extract)
dir=string
Name of direction input map (r.watershed or r.stream.extract)
multipier=integer
Multipier to store stream index value
Must be > 0
Default: 1000
cells=string
File to store pixel's position
lengths=string
File to store current stream length
