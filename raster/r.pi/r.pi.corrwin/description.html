DESCRIPTION

This module computes the correlation between two raster files but unlike r.pi.lm for moving windows of a specific size. This module used code sniplets from r.neighbours and r.covar. The program will be run non-interactively if the user specifies program arguments (see OPTIONS) on the command line. Alternately, the user can simply type r.pi.corrwin on the command line, without program arguments. In this case, the user will be prompted for flag settings and parameter values.
