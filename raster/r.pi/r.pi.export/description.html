DESCRIPTION

The r.pi.export module computes statistics of values within a patch and exports them to an ascii file. Additionally the raw patch values split up by patch ID are exported. The patch ID raster can be generated in order to visually link patch statistics and individual patches.

The program will be run non-interactively if the user specifies program arguments (see OPTIONS) on the command line. Alternately, the user can simply type r.pi.export on the command line, without program arguments. In this case, the user will be prompted for flag settings and parameter values.

NOTES

For export and import of patch values after e.g. R calculation r.pi.export and r.pi.import can be applied using the same raster (same extent, resolution is mandatory).
