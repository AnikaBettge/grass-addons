DESCRIPTION

r.soils.texture define soils texture from sand and clay distribution grid and a texture scheme definition. 
Sand and clay grid are simple grid theme; scheme file  is text ASCII file with *.dat extention like USDA.dat, FAO.dat and international.dat supplied with source code module in scheme directory.
