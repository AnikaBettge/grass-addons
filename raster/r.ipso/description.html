DESCRIPTION

r.ipso produces the Ipsometric and Ipsographic curve related to a digital elevation model and prints the percentiles.

NOTES

This curve gives the distribution of surfaces in different altitude ranges. Each point on the curve has on the y-axis the elevation and on the x-axis the portion of the basin surface placed above such elevation value. The ipsometric curve uses adimensional axes.

Flags:

-a : generates ipsometric curve

-b : generates iposographic curve

EXAMPLE

To generate iposographic curve 

r.ipso.py -b map=elevation.10m image=path/to/output/prefix

To generate ipsometric curve 

r.ipso.py -a map=elevation.10m image=path/to/output/prefix
