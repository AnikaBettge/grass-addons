DESCRIPTION

r.pack.mat packs up a GRASS raster map and related support files (color table etc) for copying to another computer as a single compressed binary MAT-File (gzip compression).

EXAMPLE

This command saves the raster map "elevation.dem" into the export file "elevation.dem.pack" which can be transferred to another computer: 
 
r.pack.mat elevation.dem

SEE ALSO

r.unpack.mat
